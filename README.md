# 📊 Deaf Education Intelligence System (DEIS)

> 🚀 AI-powered platform to analyze, predict, and improve learning outcomes for Deaf & Hard-of-Hearing students.

---

## 🌍 Problem

Deaf students face major challenges in education systems worldwide:

* Limited real-time understanding in lectures
* No personalized learning insights
* Teachers lack visibility into student struggles
* Existing tools provide accessibility, but **no intelligence**

> ❌ Accessibility exists. but insight and personalization are missing.

---

## 💡 Solution

**DEIS (Deaf Education Intelligence System)** bridges this gap by combining AI, analytics, and accessibility:

* 📊 Tracks student performance & engagement
* 🧠 Predicts learning outcomes
* 🎯 Detects weak topics automatically
* 🤖 Recommends personalized learning strategies
* 👨‍🏫 Provides actionable insights for teachers

## 🔥 Core Features

### 🧠 AI Prediction Engine

* Predicts student performance (risk detection)
* Built using Scikit-learn

---

### 🎯 Weak Topic Detection

* Identifies subjects where students struggle
* Topic-level intelligence (not just scores)

---

### 🤖 AI Recommendations

* Personalized learning suggestions powered by OpenAI
* Example:

  * “Focus on Algebra basics”
  * “Watch visual explanation”

---

### 👨‍🏫 Teacher Dashboard

* Class performance overview
* At-risk students
* Data-driven teaching insights

---

### 🧑‍🎓 Student Dashboard

* Personal progress tracking
* Weak areas
* AI-generated recommendations

---

### 💬 AI Tutor Memory

* Stores learning interactions
* Enables contextual guidance across sessions

---

### 📤 Export Reports (CSV/PDF)

* Teachers can download class reports
* Useful for analysis & sharing

---

### 💙 Encouragement System

* Teachers can send motivational messages to at-risk students

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/deaf-education-intelligence
cd deaf-education-intelligence
```

---

### 2️⃣ Backend setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

---

### 3️⃣ Frontend setup

```bash
cd frontend
npm install
npm run dev
```
